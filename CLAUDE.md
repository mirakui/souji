## Language

The official language of this repository is English. All of the following
must be written in English:

- PR titles and descriptions
- PR review comments and replies
- Issue titles, descriptions, and comments
- Commit messages
- Code comments and docstrings
- Documentation (README, specs, design docs, etc.)
- Identifiers (variable / function / class / file names)

This rule applies even when the user instructs Claude in Japanese: Claude
may converse with the user in Japanese, but any artifact that lands in the
repository or on GitHub must be in English.
